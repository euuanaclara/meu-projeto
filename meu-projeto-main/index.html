Sobre a Escola

Como sucessor do Porsche 356, o 911 conquistou, desde o início, os corações dos fãs de automóveis desportivos. O primeiro 911 surgiu em 1963, no IAA de Frankfurt, como modelo 901. A renomeação para 911 foi realizada em 1964, para o lançamento no mercado.
